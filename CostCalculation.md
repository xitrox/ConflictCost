# Konfliktkosten-Analyse

Dieses Dokument dient als Vorlage zur strukturierten Erfassung und Bewertung von Konfliktkosten in Organisationen.  
Es kann sowohl für interne Analysen als auch für Beratungszwecke genutzt werden.

---

## 1. Basisdaten

| Feld | Beschreibung | Beispiel |
|------|---------------|-----------|
| **Konflikt-ID** | Eindeutige Kennung | K2025-001 |
| **Beteiligte Parteien** | Mitarbeiter, Teams oder Abteilungen | Vertrieb ↔ Marketing |
| **Startdatum** | Beginn des Konflikts | 2025-01-15 |
| **Enddatum** | (geplant oder tatsächlich) | 2025-03-30 |
| **Dauer (Tage)** | Automatisch berechnet | 75 |
| **Bearbeitungsverantwortlicher** | HR, Konfliktmanager etc. | Julia Meier |

| **Externe Berater / Mediatoren** | Honorare, Reisekosten etc. | tatsächliche Kosten | |

| **Abfindungen / Kündigungskosten** | bei Konfliktfolgen | tatsächliche Kosten | |

| **Fluktuationskosten** | Ersatz & Einarbeitung neuer Mitarbeiter | HR-Daten | |

## 5. Ergänzende Faktoren

- **Konfliktursache:** Kommunikation, Macht, Ressourcen, Werte, Rollen etc.  
- **Konfliktintensität:** niedrig / mittel / hoch  
- **Lösungsart:** Mediation, Coaching, Führungseingriff, Trennung etc.  
- **Langfristige Wirkung:** Teamklima, Vertrauen, Leistungsfähigkeit
